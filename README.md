# MemoDrops Monorepo (v0.1)

Monorepo inicial do MemoDrops com estrutura para:

- `apps/backend` – API Fastify (Node/TS)
- `apps/web` – Frontend web (Next.js ou outro framework React)
- `apps/mobile` – App mobile (Expo/React Native – esqueleto)
- `apps/scrapers` – Scrapers de editais e bancas
- `apps/ai` – Serviços de IA (pipelines, workers)
- `packages/shared` – Tipos, utilitários e contratos compartilhados
- `docs` – Documentação da arquitetura

Este repo foi gerado como base. Os módulos avançados (SRS, IA Tutor, ReccoEngine, etc.)
devem ser encaixados aqui a partir dos blocos que já construímos.
